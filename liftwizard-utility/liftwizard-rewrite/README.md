# Liftwizard OpenRewrite Recipes

[OpenRewrite](https://docs.openrewrite.org/) recipes for automated Java code refactoring. These recipes help migrate your Java project to [Eclipse Collections](https://eclipse.dev/collections/), improve existing Eclipse Collections code with best practices, and transform logging to use SLF4J parameterized format.

## Documentation

- [Overview](https://liftwizard.io/openrewrite/overview) — composite recipes and what they do
- [Getting Started](https://liftwizard.io/openrewrite/getting-started) — Maven plugin configuration and running recipes
- [Recipe Reference](https://liftwizard.io/openrewrite/recipe-reference) — complete listing of all recipes with transformation examples
